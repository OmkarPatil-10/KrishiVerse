# KrishiVerse Frontend

A React + Vite frontend for the KrishiVerse platform.

## Features
- User authentication (login/signup)
- Farmer, buyer, contractor, and admin dashboards
- Contract management and marketplace views
- Notifications, profile settings, and wallet integration
- AI chatbot and price prediction tools

## Tech Stack
- React
- Vite
- Tailwind CSS
- Context API for auth/profile state
- Axios for API requests

## Setup
```bash
npm install
npm run dev
```

## Useful Commands
```bash
npm run build
npm run lint
```

## Project Structure
- `src/pages` — main screens
- `src/components` — reusable UI components
- `src/context` — auth/profile context
- `src/services` — API and ML integration services
- `src/i18n` — language support
